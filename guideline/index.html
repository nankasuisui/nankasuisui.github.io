・要約

個人やグループでの非商用利用(公開/非公開問わず)や、動画投稿・ライブ配信における利用については、基本的に許諾なしで自由に利用していただいて大丈夫です。
その他お金の受け渡しや音楽ファイル自体の再配布が必要な利用については、事前に問い合わせていただければだいたいの場合はOKです。
利用制限リストの曲を個人利用(非営利)の範囲を超えてインターネット上などで一般公開する形で利用されたい場合は、一度問い合わせてみてもらえれば幸いです。

・個人利用(非営利)の場合

個人/グループでの利用については、金銭が伴わない形の利用であれば事前の許諾なしにご利用が可能です。

利用想定例：

  • 個人や団体で開催する集会やイベント,TRPGセッション等のBGMとして当サークル楽曲を利用する
  • グループ内や学内にて発表する作品のBGMとして当サークル楽曲を利用する
  • その他一般的な私的利用の範囲で当サークル楽曲を利用する
  • ...

・ライブ配信・動画投稿などにおける利用の場合

Youtube, niconico等のコンテンツ投稿サイトにおける動画コンテンツ及びライブ配信のBGMとしての用途についても、事前の許諾なしにご利用が可能です。(任意でご利用報告を頂けると助かります。)

また配信や動画の収益化の有無は問いません。(動画ファイル自体を直接販売する場合を除く)

利用想定例：

  • BGMとして当サークル楽曲を利用した動画を制作し、Youtube等に投稿する
  • BGMとして当サークル楽曲を利用してライブ配信を行う
  • ...

・ゲーム内のカスタムコンテンツにおける利用の場合

ゲーム内でユーザーが制作できるカスタムコンテンツの一部として当サークル楽曲を利用される場合、ユーザーが制作したコンテンツと同時に公開する場合であれば事前の許諾なしで再配布を含む形で利用していただいても構いません。

利用想定例：

  • リズムゲーム内で当サークル楽曲を利用してカスタム譜面/レベルを作成し、Steamワークショップ等で公開する
  • BGMに当サークル楽曲を利用したワールドをVRchatで公開する
  • ...

・その他の利用の場合

その他の金銭の授受と楽曲ファイルの再配布が伴う利用については個別にご対応を行いますので、ページ下部のフォームまたは uytrere00302[at]gmail.com 宛に利用目的を添えて事前にご連絡をお願いいたします。(基本的には許諾する方向ですので、お気軽にお問い合わせください。)

利用想定例：

  • 当サークル楽曲を利用した音声・動画などを販売する
  • 制作ゲーム内で当サークル楽曲を利用し、頒布作品に音声ファイルを同梱した形で公開・販売する
  • ...

・クレジット表記について

利用に際しては、動画キャプションや利用作品内などのわかりやすい場所に クレジットやサイトへのリンクを出来るだけ表記して頂けますと幸いです。

  • サークル名かアーティスト名のほか、もし可能であればサイトや作品販売ページへのリンクも記載して頂けると大変嬉しく思います。

・禁止事項

楽曲の利用にあたり、以下に該当する行為は控えて頂けますようお願いいたします。

  1. 法令及び公序良俗に過度に反するコンテンツ・過度に反社会的な表現を含むコンテンツに利用する行為。
  2. 当サークルおよび第三者の権利を侵害する行為。
  3. 当サークルの許可を得ず楽曲データを再配布・販売する行為。
  4. 当サークル及び当サークル楽曲、または第三者の名誉・人格・思想・信条等を毀損したり不利益を与える行為。
  5. 楽曲に原曲を認識できないほどの著しい改変を加えて利用する行為。
    • カット・ループ・フェードなどの編集や、二次創作としてのリミックス・アレンジについては行っていただいて問題ありません。
  6. その他本ページ記載のガイドラインを逸脱した利用行為。

・その他注意事項

  • アルバム未収録楽曲について
    • アルバムに収録されていない楽曲のうち、bandcamp, soundcloud等でダウンロード可能に設定されている楽曲についてはアルバム収録曲と同様の条件で利用可能です。
    • その他の楽曲の利用に関しては個別にお問い合わせをお願いします。
  • 当サークル楽曲を利用する事により生じたいかなるトラブルや損害についても、当サークルはその責を負いかねますので予めご了承ください。
